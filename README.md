# DSA Solutions

This repository contains my solutions to **LeetCode** problems.

## 📌 About

* Solutions are written as I solve problems on LeetCode.
* Each file is named using the **LeetCode question number**.
* The repository is intended for practice, revision, and interview preparation.

## 📂 Structure

Each solution follows a simple naming convention:

```text
1.cpp
2.cpp
121.cpp
704.cpp
...
```

*(Replace `.cpp` with your programming language extension if applicable.)*

## 🎯 Purpose

* Track my DSA practice
* Maintain a collection of solved problems
* Use as a quick reference for future revision

## ⚠️ Note

The repository contains only the solution code. Problem statements are **not** included due to LeetCode's copyright policy.
